# 下書き結衣（Dify Advanced Chat）

## 背景
事業部のAI活用推進係に任命された際、
「AIで何かしたいがどこから始めればいいかわからない」という声が
メンバーのネックになっていると推測した。

## 解決
ツールのアイデアを自由入力するだけで、
AIが要件定義のたたき台を生成するチャット型アシスタントを構築。
AI活用の最初の一歩を下げることを目的とした。

## 機能
- 背景・課題・理想を入力として受け取り（すべて任意）
- 5W1Hの観点でヒアリングを最大3回実施
- ユーザーの指示で要件定義書を出力

## 出力
- プロジェクト概要（Why & What）
- 業務フロー（Who / When / Where）
- 機能要件詳細リスト（テーブル形式）
- AIからの推奨アドバイス

## 使用技術
- Dify（Advanced Chat）
- GPT-5
